Aplicando términos y condiciones

Resúmen

  • Las negritas se usan con <b> o <strong>
  • Las cursivas se usan con <i> o <em>
  • El subrayado se usa con <u>
  • El tachado se usa con <s>
  • El resaltado se usa con <mark>
  • El texto performateado se usa con <pre>
  • El código se usa con <code>
  • Para letras pequeñas usamos <small>
  • Para subíndices usamos <sub> y para superíndices <sup>
